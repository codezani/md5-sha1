# MD5 & SHA1 Hash Generator

A simple, fast, and **offline** web tool to generate MD5 and SHA1 hashes from any text. Built with pure HTML, CSS, and JavaScript — no backend, no data sent to any server.

![Demo](https://i.imgur.com/your-image-link.png)

## ✨ Features

- Real-time hash generation (as you type)
- Shows both **lowercase** and **UPPERCASE** versions
- Clean and modern blue & white design
- Fully client-side (works offline)
- Responsive design
- Persian (RTL) language support

## 🛠 Technologies

- HTML5
- CSS3 (modern gradients + responsive)
- Pure Vanilla JavaScript
- Reliable MD5 & SHA1 implementations

## 🚀 How to Use

1. Clone or download the project
2. Open `index.html` in any web browser
3. Type or paste your text
4. Hashes (MD5 & SHA1) will appear instantly

No installation required.
